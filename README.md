# OIBSIP – Data Science Internship

This repository contains the projects completed as part of the **AICTE Oasis Infobyte (OIBSIP) Data Science Internship**.  
The internship focused on applying data analysis and machine learning techniques to solve real-world problems using Python.

Each task follows a structured data science workflow including data preprocessing, exploratory data analysis (EDA), model building, evaluation, and result interpretation.

---

## Tasks Completed

- **Task 2:** Unemployment Analysis using Python  
- **Task 3:** Car Price Prediction using Machine Learning  
- **Task 4:** Email Spam Detection using Machine Learning  
- **Task 5:** Sales Prediction using Python  

---

## Tech Stack & Tools

- **Programming Language:** Python  
- **Libraries:** NumPy, Pandas, Scikit-learn  
- **Visualization:** Matplotlib, Seaborn  
- **Development Environment:** Jupyter Notebook  
- **Version Control:** Git & GitHub
